## 🛠 백엔드 기술 스택
> **Backend** : JAVA 11, Spring Boot (Gradle), Spring Data JPA, Spring Security, JWT, Firebase Cloud Messaging<br/>
> **Deploy** : GCP Compute Engine (CentOS), GCP SQL (MySQL), AWS S3<br/>
